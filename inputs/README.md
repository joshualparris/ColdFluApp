# Inputs

This directory contains unverified historical material used to understand the project or discover candidate claims.

- `conversations/` contains discussion and context exports.
- `prompts/` contains research prompts.
- `imported-research/` contains AI or third-party research drafts.

Nothing here is canonical, clinically reviewed, or safe to publish directly. Do not edit raw imports; create a derived research record under `content/research/` and retain provenance.
