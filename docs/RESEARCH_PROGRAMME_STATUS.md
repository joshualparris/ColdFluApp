# Research programme status

This is the provider-neutral execution view for the 28-module catalogue. `docs/ASSIGNMENTS.md` preserves historical and optional external destinations; this file records what the repository actually contains.

## Status definitions

- **Verified pre-review research:** exact reusable sources and structured extraction exist, but no human review is claimed.
- **Protocol/source orientation:** protocol and candidate-source work exist; systematic extraction is incomplete.
- **Discovery only:** imported material or an assignment exists, but no verified canonical research package exists.
