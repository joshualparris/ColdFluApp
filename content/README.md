# Canonical content

Only structured, traceable project content belongs here.

- `modules/` contains reader-facing module data.
- `sources/` contains exact reusable source records.
- `jurisdictions/` contains reviewed country or region overlays.
- `research/` contains protocols, searches, extractions, and review evidence.

This directory is currently intentionally empty except for retention markers. Existing imported research must pass the workflow in `docs/RESEARCH_AND_REVIEW.md` before entering it.
